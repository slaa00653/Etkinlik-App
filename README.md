Etkinlik Kayıt ve Takip Sistemi

Bu proje, kurum içi veya genel etkinliklerin dijital ortamda yönetilmesi, katılımcı kayıtlarının alınması ve takibinin yapılması amacıyla geliştirilmiş bir **Web Uygulamasıdır.** Proje, modern yazılım geliştirme prensiplerine uygun olarak **MVC (Model-View-Controller)** mimarisi ile inşa edilmiştir.

 Kullanılan Teknolojiler ve Araçlar

* **Dil:** C# (.NET Core)
* **Mimari:** ASP.NET Core MVC
* **Veritabanı:** MySQL (Entity Framework Core)
* **Frontend:** HTML5, CSS3, Bootstrap, JavaScript & jQuery
* **Versiyon Kontrol:** Git & GitHub

 Temel Özellikler

* **Dinamik Etkinlik Yönetimi:** Yeni etkinlikler oluşturma, mevcut olanları düzenleme ve yayından kaldırma.
* **Başvuru Modülü:** Kullanıcıların etkinliklere hızlı ve güvenli bir şekilde kayıt olabilmesi.
* **Yönetim Paneli:** Etkinlik bazlı katılımcı listelerini görüntüleme ve veritabanı senkronizasyonu.
* **Responsive Tasarım:** Hem mobil hem de masaüstü cihazlarla tam uyumlu arayüz.
 Proje Klasör Yapısı

* `Controllers/`: Uygulama mantığının ve yönlendirmelerin yönetildiği katman.
* `Models/`: Veritabanı tablolarının ve nesne modellerinin tanımlandığı katman.
* `Views/`: Kullanıcının etkileşime girdiği arayüz dosyaları.
* `wwwroot/`: CSS, JavaScript ve kütüphane gibi statik dosyaların bulunduğu klasör.
